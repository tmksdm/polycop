# Polycop

Personal desktop copy-trading tool for Polymarket.

## Status

Project is in early development.

Current stage:

- Stage 0 — environment setup and Python basics.

## Safety

This project is designed for personal use only.

Important rules:

- Never commit `.env`.
- Never commit private keys.
- Never paste real private keys into code, logs, GitHub, or chat.
- First real trades must be done only after successful DRY-RUN testing.
- First LIVE test should use very small size: $1–2, ratio 1%, limited wallet balance.
